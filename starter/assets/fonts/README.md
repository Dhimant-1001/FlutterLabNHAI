# Poppins Font Files

Download Poppins fonts from Google Fonts and place them here.

## Required files:
- `Poppins-Regular.ttf`
- `Poppins-Bold.ttf`

## How to download:
1. Go to [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins)
2. Click "Download family"
3. Extract the ZIP file
4. Copy the following files to this folder:
   - `Poppins-Regular.ttf`
   - `Poppins-Bold.ttf`

After adding the fonts, delete this README file.

# Avatar Image Placeholder

Place your avatar image here as `avatar.png`.

## Requirements:
- Square image (recommended 200x200 or larger)
- PNG format
- Name must be exactly: `avatar.png`

## Where to get images:
- Use your own photo
- Download from [UI Faces](https://uifaces.co/)
- Download from [Unsplash](https://unsplash.com/)
- Use [This Person Does Not Exist](https://thispersondoesnotexist.com/)

After adding the image, delete this README file.
